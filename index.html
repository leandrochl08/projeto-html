O inicio da Internet

(Início)

A internet surgiu com a ARPANET, a partir de um projeto de uma agência norte americana responsável por interligar quatro instituições de pesquisa. A internet como conhecemos hoje se tornou um conglomerado de redes interligadas pelo mundo, visa buscar conectividade independente do dispositivo tecnologico que esteja a ela conectada, esta conectividade se da através dos protocolos, que são responsáveis pela comunicação de dados, voz, vídeo e som através da internet.

O que é o HTML?

(Início)

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem entendida pelo computador e que está presente na maior parte dos sites e aplicativos online. Um hipertexto é um texto que faz referência a outros textos, já uma linguagem de marcação é composta por uma série instruções em marcação que comunicam aos servidores da web qual o estilo e estrutura de um documento. Não se considera o HTML uma linguagem de programação, pois não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Surgimento do HTML

(Início)

O HTML foi criado em 1991 pelo britânico cientista da computação e físico do centro de pesquisas CERN, da Suíça, Tim Berners-Lee. O código foi elaborado para servir de complemento ao World Wide Web (WWW) e ser uma forma de interligar outros textos presentes na rede através de links. A primeira versão do HTML foi publicada em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos.

Atributos de Tags

(Início)

Atributos HTML são palavras especiais utilizadas em uma tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com auxilio dos atributos, podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc. Segue alguns exemplos:

  • class="NomeDaClasse"
  • id="NomeDoId"
  • class="NomeDaClasse"
  • src="Link ou diretório da mídia"
  • href="Url"
